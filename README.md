# 🧉 Termolandia - E-commerce de Termos

Este proyecto es un Trabajo Práctico desarrollado para el programa **Talento Tech**. Consiste en una landing page de comercio electrónico para una tienda especializada en la venta de termos de alta calidad, enfocada en aplicar conceptos sólidos de maquetación web.

## 🚀 Tecnologías y conceptos aplicados:
- **HTML5:** Estructuración semántica del sitio (`<header>`, `<main>`, `<section>`, `<footer>`).
- **CSS3:** Diseño de estilos independientes y ordenados.
- **Flexbox:** Uso intensivo de modelos de caja flexible para la distribución del menú de navegación, alineación vertical de contenidos y diseño de la grilla de productos.
- **Diseño Responsive:** Implementación de propiedades como `flex-wrap`, `flex-basis` y límites de ancho (`max-width`) para garantizar que las tarjetas de productos se adapten automáticamente a diferentes tamaños de pantalla (celulares, tablets y monitores).

## 🛠️ Estructura del Proyecto:
- `index.html`: Estructura y contenido del sitio.
- `style.css`: Estilos, posicionamiento, diseño flexible y adaptabilidad.
- `/img`: Directorio de recursos visuales y multimedia utilizados.
